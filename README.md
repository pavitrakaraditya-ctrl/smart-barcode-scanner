# Smart Barcode Scanner using ESP32-S3

## Overview
This project is an embedded system-based smart barcode scanner designed for retail automation.

## Features
- Real-time barcode detection
- Product identification
- LCD display output

## Components Used
- ESP32-S3
- OV2640 Camera Module
- LCD Display

## Technologies
- Embedded C
- ESP32
- IoT

## Description
Developed firmware for ESP32-S3 to process barcode data and display output in real-time. The system is designed as a prototype for self-service billing systems.
